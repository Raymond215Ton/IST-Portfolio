What is an internet protocol? Compare HTTP, TCP and FTP.

An internet protocol is a set of rules for routing and addressing packets of data so that they can travel across networks and arrive at the correct destination. Data that traverses over the Internet are separated into smaller parts called packets. These packets have IP information attached to them and this information helps them go towards the correct destination. Once these packets arrive at their intended destination, they are sorted differently based on which transport protocol is utilised with the IP.

What is HTTP?

HTTP, also referred to as Hypertext Transfer Protocol,, is the underlying protocol for the World Wide Web and determines how messages are formatted and transmitted as well as what actions web browsers and servers should take. HTTP is a stateless protocol because each command is performed individually, with no prior knowledge of the command that came before it.

What is TCP?
